# DeviceTrackerWebService

JDK 1.8 Required<br>
JRE 8 Required

To run service enter following command:<br>
$ java -jar target/redsky_device_tracking_service-1.0-SNAPSHOT.jar 


<b>To Build:</b>
1. First install maven
http://maven.apache.org/install.html
2. Run command:
$ mvn package
3. Run service:
$ java -jar target/redsky_device_tracking_service-1.0-SNAPSHOT.jar
